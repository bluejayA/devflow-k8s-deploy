## v0.1.0 제약 (Known Limitations)

- **JVM 스택 전용** (Kotlin + Spring Boot). Go/Python/React는 v0.2~v0.4 로드맵
- **Deployment만 생성** (StatefulSet 미지원 — stateful 앱 감지 시 경고만)
- **auto-fix 루프 v0.2+ 예정** — 현재는 수동 개입 안내
- `kubeconform` 같은 offline validator 미도입 (v0.2+ 로드맵) — 현재는 `--validate=false` + connection refused graceful skip으로 우회
- BuildPlan 일반화(JDK/JRE 2단계 가정 완화) — v0.2+ Go 추가 직전
- 도움말 카탈로그 외부 파일(`help/{lang}.yml`) 분리 — v0.2+
- `NetworkPolicy` / `PodDisruptionBudget` / PVC 미생성 — v0.2+
